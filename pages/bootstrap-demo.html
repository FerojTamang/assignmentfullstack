What is Bootstrap?

Bootstrap is the world's most popular front-end open source toolkit, featuring Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful JavaScript plugins.

Responsive Design

Bootstrap's responsive CSS adjusts to phones, tablets, and desktops using CSS media queries.

Fast Development

Pre-built components and utilities help you build interfaces faster than ever before.

Customizable

Customize Bootstrap with your own CSS or by using Sass variables to fit your brand.

Community Driven

Millions of developers worldwide use Bootstrap to build amazing web experiences.

Bootstrap Features

Discover the powerful features that make Bootstrap the top choice for developers

Flexible Grid System

12-column responsive grid system that scales up to 12 columns as the device or viewport size increases.

Extensive Components

Over 20 flexible components including buttons, cards, navbars, modals, and much more.

JavaScript Plugins

Interactive components powered by JavaScript including carousels, modals, tooltips, and dropdowns.

Sass Variables

Customize Bootstrap's Sass variables to create your own unique design system and brand colors.

Utility Classes

Hundreds of utility classes for spacing, colors, typography, flexbox, and positioning.

Accessibility

Built with accessibility in mind, including proper ARIA attributes and keyboard navigation.
