What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It forms the structure of a webpage by defining headings, paragraphs, lists, tables, links, images, and other content. It can also support basic interactivity without CSS or advanced JavaScript.

Inline vs Block-level Tags

Block elements take up the full width and start on a new line, while inline elements occupy only as much space as needed and stay in line with text.

Type Example Tags Description
Block-level <div>, <p>, <h1>-<h6>, <section>, <article>, <aside>, <header>, <footer> Starts on a new line, takes full width, can contain block and inline elements.
Inline <span>, <a>, <strong>, <em>, <img>, <mark> Flows within text, occupies only content width, usually contains text or inline elements.

Important Features of HTML

  • Defines structure with headings, paragraphs, lists, tables, links, images, forms, and media.
  • Supports semantic elements like <header>, <footer>, <article>, <section>, <aside> for accessibility.
  • Includes multimedia with <audio> and <video>.
  • Enables forms and basic interactivity.
  • Controls content flow with inline and block elements.
  • Facilitates navigation with hyperlinks.

Commonly Used HTML Tags:

Tag Purpose Example
